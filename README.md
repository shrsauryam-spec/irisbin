# Iris Bin
### Automatic Mechanical Iris Bin Using Arduino Nano, Servo Motor (MG90S) and Ultrasonic Sensor

THE PROJECT DOES NOT USE A PCB. NORMAL WIRING WILL BE USED.

## Key Features:
1. You can throw trash without needing to touch the cap (so fun, ikr, more fun when you are throwing paper from far away)
2. Detachable Bottom for ease in removing trash
3. MAGNETS are used for attaching most parts

## Materials Required:
1. Servo Motor x1
2. Arduino Nano (or any other MCU of your choice)
3. 5V Power Output
4. Gears
5. Ultrasonic Sensor
